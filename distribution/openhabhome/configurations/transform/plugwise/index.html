roomidId of roomCan only be used in combination with type appliances
And will show all Appliances in room with id #

And will show all Appliances in group with id #
Example: data.xml?type=appliances&roomid=2 (will show all aplliances in room 2)


actions.html
This file contains actions that can be send to Plugwise source.

There are currently 4 options

- switchon
- switchoff
- lockon
- lockoff

all options need an appliance id.

Example: actions.html?option=switchoff&id=22 (will turn off appliance 22)
